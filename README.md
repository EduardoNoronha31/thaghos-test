  # 👨‍💻 Development

  <br>

  ## ⚙️ Run the app

  ```
  ## Clone the repository in your text editor

  git clone https://github.com/EduardoNoronha31/thaghos-test.git


  ## Install the initial project files

  yarn
  ## or
  npm install


  ## Run the project on your computer.
  yarn run dev
  ## or
  npm run dev
  ```

  ## ⚙️ Run the tests

  ```
  yarn test
  ## or
  npm test
  ```
